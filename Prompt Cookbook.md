# Prompt Cookbook

Use these prompts with Codex while this vault is open as the workspace.

## Ingest One Source

`Read raw/inbox/<filename>.md and ingest it into my personal wiki. Create or update the relevant source, concept, entity, index, and log pages.`

## Ingest Everything In Inbox

`Process every source in raw/inbox/ one by one. After each file, tell me which wiki pages changed and what new concepts or contradictions you found.`

## Batch-Ingest Lenny's Podcast

`Ingest every transcript in raw/inbox/ that is not yet reflected in the wiki. Create or update source notes, guest entity pages, dashboards, index.md, and log.md.`

## Build A Topic From Existing Notes

`Review the current wiki and build out a concept page for <topic>. Merge overlapping pages if needed and update the index.`

## Save A Query Result

`Answer this question from the wiki and save the result as a durable analysis note in wiki/analyses/: <question>`

## Lint The Wiki

`Lint this vault for duplicate pages, stale claims, missing links, orphan pages, and concepts that deserve their own page. Make high-confidence fixes and summarize the rest.`

## Weekly Review

`Review the last 7 log entries, summarize what changed in the wiki, identify the most active themes, and suggest the next three sources or questions to pursue.`

## Personal Knowledge Tracking

`Using my current wiki, summarize what it suggests about my recurring goals, bottlenecks, and unresolved questions. Save the result as an analysis note.`
