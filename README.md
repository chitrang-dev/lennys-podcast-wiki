# Personal LLM Wiki

This repository is an Obsidian vault designed to be maintained with Codex.

The workflow is:

1. Put raw source material into `raw/inbox/`.
2. Ask Codex to ingest it into the wiki.
3. Browse the resulting pages in Obsidian.
4. Ask questions against the wiki and save good answers back into it.

Start with [[Getting Started]], then open [[index]].
