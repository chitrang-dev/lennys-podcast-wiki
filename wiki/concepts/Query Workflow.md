# Query Workflow

## Summary

Questions should be answered from the maintained wiki first, then saved back into the wiki when the answer has durable value.

## Pattern

1. Start from [[index]].
2. Read the most relevant pages.
3. Synthesize an answer with explicit references to source-backed pages.
4. If the result is worth keeping, save it in `wiki/analyses/`.

## Good Saved Outputs

- Comparisons
- Decision memos
- Reading syntheses
- Research summaries
- Personal reflections grounded in prior notes
